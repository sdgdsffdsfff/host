- 地址：[host.qiang.it](http://host.qiang.it)

## 功能列表
- [x] 统一登陆账号：aotu
- [x] 业务管理
	- [x] 支持一个业务填写多个地址
	- [x] 支持 CMS 地址快捷输入
	- [] 输入地址校验
- [x] host 方案管理
	- [x] 支持添加方案描述
	- [x] 支持 host 规则批量输入
	- [] host 规则校验
